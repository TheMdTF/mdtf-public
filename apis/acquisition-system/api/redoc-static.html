Image Submission

Endpoints for submitting images to an ongoing transaction.

Associate a face image capture with the ongoing transaction.

Acquisition system providers may submit face image data through this endpoint. Submitted image captures will be stored and associated with the ongoing transaction. Multiple images may be submitted during a transaction.

Request Body schema: application/json
required

Face image collected as part of a transaction.

StationID
required
string

The unique identifier for the station that the artifact was captured at.

FaceImageData
required
string <byte>

The captured face image data in PNG format, encoded as a base64 string. The data string shall not exceed 1MB.

object (PADResult)

The result data from a PAD interaction

Responses

https://api.mdtf.org/v1/face-captures
